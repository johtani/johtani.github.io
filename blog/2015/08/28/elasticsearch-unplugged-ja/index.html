Elasticsearch unplugged - 2.0гҒ«гҒҠгҒ‘гӮӢгғҚгғғгғҲгғҜгғјгӮҜгҒ®еӨүжӣҙ(ж—Ҙжң¬иӘһиЁі)

  • гғҺгғјгғүжғ…е ұгҒ®еӨүжӣҙ

вҖ»гҒ“гҒ®иЁҳдәӢгҒҜж¬ЎгҒ®гғ–гғӯгӮ°гӮ’зҝ»иЁігҒ—гҒҹгӮӮгҒ®гҒ«гҒӘгӮҠгҒҫгҒҷгҖӮ

еҺҹж–ҮпјҡElasticsearch unplugged - Networking changes in 2.0

ElasticsearchгӮ’гғӯгғјгӮ«гғ«гҒ®гғһгӮ·гғігҒ§иө·еӢ•гҒ—гҒҫгҒҷгҖӮ гҒқгҒ—гҒҰгҖҒжҳЁж—Ҙи©ҰгҒ—гҒҹгғҮгғјгӮҝгӮ’еүҠйҷӨгҒҷгӮӢгҒҹгӮҒгҒ«DELETE *гӮ’е®ҹиЎҢгҒ—гҒҫгҒҷгҖӮ гҒҷгӮӢгҒЁгҖҒжӮІгҒ—гҒқгҒҶгҒӘеҸ«гҒігӮ’еҗҢеғҡгҒҢзҷәгҒ—гҒҰгҒ„гӮӢгҒ“гҒЁгҒ«ж°—гҒҘгҒҚгҖҒгҒӘгҒңгҒқгӮ“гҒӘгҒ“гҒЁгҒ«гҒӘгҒЈгҒҰгҒ„гӮӢгҒ®гҒӢдёҚжҖқиӯ°гҒ«жҖқгҒҶгҒ§гҒ—гӮҮгҒҶгҖӮгҖӮгҖӮ

ElasticsearchгҒҜгҒ„гҒӨгӮӮгҖҒиҰӘгҒ—гҒҝгӮ„гҒҷгҒ„гӮӮгҒ®гҒ§гҒ—гҒҹгҖӮ иӨҮж•°гғҺгғјгғүгҒ®гӮҜгғ©гӮ№гӮҝгҒҢгҒ©гҒ®гӮҲгҒҶгҒ«ж©ҹиғҪгҒҷгӮӢгҒ®гҒӢгӮ’гғҶгӮ№гғҲгҒҷгӮӢгҒ«гҒҜгҖҒ гғӯгғјгӮ«гғ«гҒ®гғһгӮ·гғігҒ§гҒ„гҒҸгҒӨгҒӢгҒ®ElasticsearchгҒ®гӮӨгғігӮ№гӮҝгғігӮ№гӮ’иө·еӢ•гҒҷгӮӢгҒ гҒ‘гҒ§гҒ—гҒҹгҖӮ иө·еӢ•гҒ—гҒҹгӮӨгғігӮ№гӮҝгғігӮ№гҒҜгғһгғ«гғҒгӮӯгғЈгӮ№гғҲгӮ’еҲ©з”ЁгҒ—гҒҰиҮӘеӢ•зҡ„гҒ«гҒҠдә’гҒ„гӮ’иҰӢгҒӨгҒ‘гҒҰгҖҒ1гҒӨгҒ®гӮҜгғ©гӮ№гӮҝгҒ«гҒӘгӮҠгҖҒиІ иҚ·гӮ’е…ұжңүгҒ—е§ӢгӮҒгҒҫгҒҷгҖӮ гҒ—гҒӢгҒ—гҖҒгҒ“гӮҢгҒҜиҰӘгҒ—гҒҝгӮ„гҒҷгҒҷгҒҺгҒҫгҒ—гҒҹгҖӮ гӮ«гғігғ•гӮЎгғ¬гғігӮ№гҒӘгҒ©гҒ§гҖҒгғӯгғјгӮ«гғ«гҒ®гғһгӮ·гғігҒ§ElasticsearchгӮ’иө·еӢ•гҒ—гҒҰгҒҝгҒҰгҒҸгҒ гҒ•гҒ„гҖӮ гҒҷгӮӢгҒЁ100гғҺгғјгғүгҒ®гӮҜгғ©гӮ№гӮҝгҒ«еҸӮеҠ гҒ—гҒҰгҒ„гӮӢгҒ®гҒҢгҒҷгҒҗгҒ«гӮҸгҒӢгӮӢгҒ§гҒ—гӮҮгҒҶгҖӮ

гӮӮгҒҶгҒҷгҒҗгғӘгғӘгғјгӮ№гҒ•гӮҢгӮӢгҖҒ2.0.0-beta1гҒ§гҒҜгҖҒElasticsearchгҒҢйҖҡдҝЎе…ҲгӮ’йҒёжҠһгҒҷгӮӢгғҚгғғгғҲгғҜгғјгӮҜгҒ®ж©ҹиғҪгҒ«й–ўгҒҷгӮӢеӨүжӣҙгҒҢгҒӮгӮҠгҒҫгҒҷгҖӮ гҒҹгҒ гҒ—гҖҒгҒ“гӮҢгҒҫгҒ§йҖҡгӮҠгҖҒз°ЎеҚҳгҒ«й–ӢзҷәиҖ…гҒҢзөҢйЁ“гҒ§гҒҚгӮӢж©ҹиғҪгӮӮж®ӢгҒЈгҒҰгҒ„гҒҫгҒҷгҖӮ

localhostгҒёгҒ®гғҗгӮӨгғігғү

д»ҘеүҚгҖҒElasticsearchгҒҜгғҮгғ•гӮ©гғ«гғҲгҒ§гҖҒеҲ©з”ЁеҸҜиғҪгҒӘгғҚгғғгғҲгғҜгғјгӮҜгӮӨгғігӮҝгғ•гӮ§гғјгӮ№е…ЁгҒҰгҒ«гғҗгӮӨгғігғүгҒ—гҒҰгҒ„гҒҫгҒ—гҒҹгҖӮ гҒқгҒ“гҒӢгӮүгҖҒдёҖз•ӘйҒ©гҒ—гҒҹгӮӨгғігӮҝгғ•гӮ§гғјгӮ№гӮ’publish_hostгҒЁгҒ—гҒҰйҒёжҠһгҒ—гӮҲгҒҶгҒЁгҒ—гҒҫгҒҷгҖӮ гҒ“гҒ®гӮўгғүгғ¬гӮ№гҒҜElasticsearchгҒҢгӮҜгғ©гӮ№гӮҝгҒ®д»–гҒ®гғҺгғјгғүгҒЁгӮ„гӮҠгҒЁгӮҠгҒҷгӮӢгҒҹгӮҒгҒ®гӮўгғүгғ¬гӮ№гҒ§гҒҷгҖӮ

Elasticsearch 2.0гҒ§гҒҜгҖҒгғҮгғ•гӮ©гғ«гғҲгҒ§гҒҜгҖҒlocalhostгҒ«гҒ®гҒҝгғҗгӮӨгғігғүгҒ—гҒҫгҒҷгҖӮ 127.0.0.1пјҲIPv4пјүгҒЁ[::1]пјҲIPv6пјүгҒ®дёЎж–№гҒ«гғҗгӮӨгғігғүгҒ—гӮҲгҒҶгҒЁгҒ—гҒҫгҒҷгҖӮ гҒҫгҒҹгҖҒгҒ©гҒЎгӮүгҒӢгҒ®гҒҝгҒ®з’°еўғгҒ§гӮӮеӢ•дҪңгҒ—гҒҫгҒҷгҖӮ гҒ“гҒ®еӨүжӣҙгҒҜгҖҒзү№гҒ«жҢҮе®ҡгҒҢгҒӘгҒ„йҷҗгӮҠгҖҒElasticsearchгҒҢгғҚгғғгғҲгғҜгғјгӮҜдёҠгҒ®д»–гҒ®гғҺгғјгғүгҒЁжҺҘз¶ҡгҒ—гҒҫгҒӣгӮ“гҖӮ жң¬з•Әз’°еўғгҒ«з§»иЎҢгҒҷгӮӢе ҙеҗҲгҒҜгҖҒnetwork.hostгғ‘гғ©гғЎгғјгӮҝгӮ’дҪҝгҒЈгҒҰиЁӯе®ҡгҒ—гҒҫгҒ—гӮҮгҒҶгҖӮ иЁӯе®ҡгҒҜгҖҒelasticsearch.ymlгҒ«иЁҳиҝ°гҒҷгӮӢгҒӢгҖҒгӮігғһгғігғүгғ©гӮӨгғігҒ§жҢҮе®ҡгҒ—гҒҫгҒҷгҖӮ

bin/elasticsearch --network.host 192.168.1.5
bin/elasticsearch --network.host _non_loopback_

network.hostгҒ®е…ЁгҒҰгҒ®гӮӘгғ—гӮ·гғ§гғігҒ«гҒӨгҒ„гҒҰгҒҜгҖҒnetwork settingsгҒ®гғүгӮӯгғҘгғЎгғігғҲгӮ’гҒ”гӮүгӮ“гҒҸгҒ гҒ•гҒ„гҖӮ

гғһгғ«гғҒгӮӯгғЈгӮ№гғҲгҒҜе»ғжӯў

Elasticsearch 1.xгҒҜгғҚгғғгғҲгғҜгғјгӮҜгҒ®д»–гҒ®гғҺгғјгғүгҒ«жҺҘз¶ҡгғ»жҺўзҙўгҒҷгӮӢгҒҹгӮҒгҒ«гғһгғ«гғҒгӮӯгғЈгӮ№гғҲгӮ’дҪҝз”ЁгҒ—гҒҫгҒ—гҒҹгҖӮ гғһгғ«гғҒгӮӯгғЈгӮ№гғҲгҒҜйӯ”жі•гҒ®гӮҲгҒҶгҒӘжҢҷеӢ•гҒ§гҒҷгҖӮгҖӮгҖӮ ж®ӢеҝөгҒӘгҒҢгӮүгҖҒгғһгғ«гғҒгӮӯгғЈгӮ№гғҲгҒ®гӮөгғқгғјгғҲгҒҜиүҜгҒҸгӮӮжӮӘгҒҸгӮӮгҒӮгӮҠгҒҫгҒҷгҖӮ LinuxгҒҜгғӯгғјгӮ«гғ«гғӣгӮ№гғҲгҒ§гғһгғ«гғҒгӮӯгғЈгӮ№гғҲгҒ®еҫ…гҒЎеҸ—гҒ‘гӮ’гҒ—гҒҰгҒ„гҒҫгҒӣгӮ“гҖӮ OS/XгҒҜж§ӢжҲҗгҒ•гӮҢгҒҹгӮўгғүгғ¬гӮ№гҒ®е…ЁгҒҰгҒ®гӮӨгғігӮҝгғ•гӮ§гғјгӮ№гҒ«гғһгғ«гғҒгӮӯгғЈгӮ№гғҲгҒ§й…ҚдҝЎгҒ§гҒҚгҒҫгҒҷгҖӮ гҒҫгҒҹгҖҒгғҚгғғгғҲгғҜгғјгӮҜгҒ«гӮҲгҒЈгҒҰгҒҜгғһгғ«гғҒгӮӯгғЈгӮ№гғҲгҒҜгғҮгғ•гӮ©гғ«гғҲгҒ§гҒҜдҪҝз”ЁгҒ§гҒҚгҒӘгҒҸгҒӘгҒЈгҒҰгҒ„гҒҫгҒҷгҖӮ

Elasticsearch 2.0гҒҜз•°гҒӘгӮӢгӮўгғ—гғӯгғјгғҒгӮ’жҺЎз”ЁгҒ—гҒҫгҒ—гҒҹгҖӮ гғһгғ«гғҒгӮӯгғЈгӮ№гғҲгӮ’е»ғжӯўгҒ—гҒҫгҒҷпјҲгҒҹгҒ гҒ—гҖҒж–°гҒҹгҒ«гғ—гғ©гӮ°гӮӨгғігҒЁгҒ—гҒҰжҸҗдҫӣгҒ—гҒҫгҒҷпјүгҖӮ д»ЈгӮҸгӮҠгҒ«гҖҒгғӯгғјгӮ«гғ«гғӣгӮ№гғҲгҒ§гҒҜгҖҒElasticsearchгҒҜtransport.tcp.portгҒ§жҢҮе®ҡгҒ•гӮҢгҒҰгҒ„гӮӢзҜ„еӣІпјҲгғҮгғ•гӮ©гғ«гғҲгҒҜ9300-9400пјүгҒ®жңҖеҲқгҒ®5гғқгғјгғҲгҒ«еҜҫгҒ—гҒҰгғҰгғӢгӮӯгғЈгӮ№гғҲгӮ’дҪҝз”ЁгҒ§гҒҚгӮӢгӮҲгҒҶгҒ«гҒ—гҒҫгҒҷгҖӮ

гҒ“гӮҢгҒҜгҖҒй–ӢзҷәиҖ…гҒ®гҒҹгӮҒгҒ®гҖҒиЁӯе®ҡгҒҷгӮӢгҒ“гҒЁгҒӘгҒҸиҮӘеӢ•зҡ„гҒ«гӮҜгғ©гӮ№гӮҝгӮ’зө„гӮҖгҒЁгҒ„гҒҶж©ҹиғҪгӮ’ж®ӢгҒ—гҒҰгҒ„гҒҫгҒҷгҖӮ гҒ—гҒӢгҒ—гҖҒжң¬з•ӘгҒ«з§»иЎҢгҒҷгӮӢгҒЁгҒҚгҒҜгҖҒunicast hostsгҒ§ж¬ЎгҒ®гӮҲгҒҶгҒ«гғӘгӮ№гғҲгӮ’жҢҮе®ҡгҒҷгӮӢеҝ…иҰҒгҒҢгҒӮгӮҠгҒҫгҒҷгҖӮ

discovery.zen.ping.unicast.hosts: [ 192.168.1.2,  192.168.1.3 ]

unicast hostsгҒЁгҒ—гҒҰгӮҜгғ©гӮ№гӮҝгҒ«гҒӮгӮӢгғҺгғјгғүгҒ®е…ЁгҒҰгҒ®гғӘгӮ№гғҲгӮ’жҢҮе®ҡгҒҷгӮӢеҝ…иҰҒгҒҜгҒӮгӮҠгҒҫгҒӣгӮ“гҖӮ е°‘гҒӘгҒҸгҒЁгӮӮгҖҒгғһгӮ№гӮҝгғҺгғјгғүгҒЁгҒ—гҒҰйҒёеҮәгҒ•гӮҢгӮӢгҒ№гҒҚгӮӮгҒ®гӮ’жҢҮе®ҡгҒ—гҒҫгҒҷгҖӮ е·ЁеӨ§гҒӘгӮҜгғ©гӮ№гӮҝгҒ§гҒҜгҖҒ3гҒӨгҒ®е°Ӯз”ЁгҒ®гғһгӮ№гӮҝгғҺгғјгғүгӮ’жҢҒгҒЈгҒҰгҒҠгӮҠгҖҒгҒ“гҒ®3гҒӨгӮ’unicast hostsгҒЁгҒ—гҒҰиЁӯе®ҡгҒҷгӮӢгҒ“гҒЁгӮ’жҺЁеҘЁгҒ—гҒҰгҒ„гҒҫгҒҷгҖӮ

гҒ“гӮҢгҒ«гӮҲгӮҠгҖҒй–ӢзҷәгҒ®зҹҘиӯҳгғ»зөҢйЁ“гҒҢгҖҒз§ҒгҒҹгҒЎгҒ®жҺЁеҘЁгҒҷгӮӢжң¬з•ӘгҒ§гҒ®гғҚгғғгғҲгғҜгғјгӮҜиЁӯе®ҡгҒ«гҖҒгӮҲгӮҠиҝ‘гҒ„гӮӮгҒ®гҒЁгҒӘгӮҠгҒҫгҒҷгҖӮ

гғҺгғјгғүжғ…е ұгҒ®еӨүжӣҙ

жңҖеҫҢгҒ«гҖҒinet[/127.0.0.1:9200]гҒЁгҒ„гҒЈгҒҹгӮ·гғігӮҝгғғгӮҜгӮ№гӮ’е»ғжӯўгҒ—гҒҫгҒҷгҖӮ гҒ“гӮҢгҒҜгҖҒElasticsearchгҒҢnodes-info APIгҒӘгҒ©гҒ§гҖҒдҪҝз”ЁгҒ—гҒҰгҒ„гҒҹIPгӮўгғүгғ¬гӮ№гҒ®гҒҹгӮҒгҒ®гӮ·гғігӮҝгғғгӮҜгӮ№гҒ§гҒҷгҖӮ д»ҠгҒҜгҖҒRFCгҒ«жә–жӢ гҒ—гҒҹеҪўгҒ§иЎЁзӨәгҒ—гҒҫгҒҷгҖӮ 127.0.0.1:9200пјҲIPv4пјүгӮ„[::1]:9200пјҲIPv6пјүгҒ®гӮҲгҒҶгҒ«гҒ§гҒҷгҖӮ

иіӘе•ҸгҒҢгҒӮгӮӢе ҙеҗҲгҒҜгҖҒElasticsearchгҒ®Webгғ•гӮ©гғјгғ©гғ гҒ§иіӘе•ҸгҒ—гҒҰгҒҸгҒ гҒ•гҒ„гҖӮгғҷгғјгӮҝгҒҜгӮӮгҒҶгҒҷгҒҗгҒ§гҒҷпјҒпјҲзҝ»иЁігҒ—гҒҹжҷӮзӮ№гҒ§гҖҒгҒҷгҒ§гҒ«гғҷгғјгӮҝгғӘгғӘгғјгӮ№гҒ•гӮҢгҒҰгҒ„гҒҫгҒҷгҖӮпјү
