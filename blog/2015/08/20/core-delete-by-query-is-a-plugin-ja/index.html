вҖ»гҒ“гҒ®иЁҳдәӢгҒҜж¬ЎгҒ®гғ–гғӯгӮ°гӮ’зҝ»иЁігҒ—гҒҹгӮӮгҒ®гҒ«гҒӘгӮҠгҒҫгҒҷгҖӮ

еҺҹж–ҮпјҡThe Delete by Query API Is now a plugin

ElasticsearchгҒ®2.0.0-beta1гҒ§гҒҜгҖҒгҒ“гӮҢгҒҫгҒ§гҒӮгҒЈгҒҹ Delete by Query APIгҒҢеүҠйҷӨгҒ•гӮҢгҖҒ ж–°гҒ—гҒҸ Delete by Query pluginгҒ«зҪ®гҒҚжҸӣгҒҲгӮүгӮҢгҒҰгҒ„гҒҫгҒҷгҖӮ

гӮӮгҒ—гҖҒDelete by Query гӮ’еҲ©з”ЁгҒҷгӮӢе ҙеҗҲгҖҒ2.0гҒ«гӮўгғғгғ—гӮ°гғ¬гғјгғүгҒ—гҒҹгҒӮгҒЁгҒҜгҖҒгғ—гғ©гӮ°гӮӨгғігӮ’гӮӨгғігӮ№гғҲгғјгғ«гҒ—гҖҒгғүгӮӯгғҘгғЎгғігғҲгҒ«еҫ“гҒЈгҒҰгҒҸгҒ гҒ•гҒ„гҖӮ

bin/plugin install delete-by-query

гҒӘгҒңгғ—гғ©гӮ°гӮӨгғігҒ«пјҹ

ElasticsearchгҒ®гӮігӮўгҒӘAPIгҒ®е“ҒиіӘгӮ’дҝқгҒӨгҒҹгӮҒгҒ§гҒӮгӮҠгҖҒд»ҘеүҚгҒ®Delete by QueryгҒ®е®ҹиЈ…гҒҜз°ЎеҚҳгҒ«гҒҜгғ•гӮЈгғғгӮҜгӮ№гҒ§гҒҚгҒӘгҒ„еӨ§гҒҚгҒӘе•ҸйЎҢгҒҢгҒӮгӮҠгҒҫгҒ—гҒҹгҖӮ

  • еҗ„гғӘгӮҜгӮЁгӮ№гғҲгҒ®гҒӮгҒЁгҒ«гҖҒrefreshгӮ’е®ҹиЎҢгҒ—гҒҫгҒҷгҖӮгҒ“гӮҢгҒҜгҖҒеүҠйҷӨгҒ•гӮҢгҒҹгғҮгғјгӮҝгҒҢжғіе®ҡеӨ–гҒ«жӨңзҙўгҒ«еҮәгҒҰгҒ“гҒӘгҒ„гӮҲгҒҶгҒ«гҒҷгӮӢгҒҹгӮҒгҒ§гҒҷгҖӮ
    гҒҫгҒҹгҖҒгӮ»гӮ°гғЎгғігғҲгҒҢеӨ§йҮҸгҒ«гҒ§гҒҚгҖҒгғһгғјгӮёгҒҢеӨ§йҮҸгҒ«зҷәз”ҹгҒ—гҖҒгғ’гғјгғ—гҒҢеӨ§йҮҸгҒ«ж¶ҲиІ»гҒ•гӮҢгҒҰгӮӨгғігғҮгӮӯгӮ·гғігӮ°гҒҢеҠҮзҡ„гҒ«гӮ№гғӯгғјгғҖгӮҰгғігҒ—гҖҒгӮҜгғ©гӮ№гӮҝгҒ®иӨҮж•°гҒ®гғҺгғјгғүгҒҢгӮҜгғ©гғғгӮ·гғҘгҒ—гҒҰгҒ—гҒҫгҒҶзҠ¶жіҒгӮӮеј•гҒҚиө·гҒ“гҒ—гҒҫгҒ—гҒҹгҖӮ
  • гҒ“гҒ®гӮҜгӮЁгғӘгҒҜгҖҒгғ—гғ©гӮӨгғһгғӘгҖҒгғ¬гғ—гғӘгӮ«гҒ®дёЎж–№гҒ§е®ҹиЎҢгҒ•гӮҢгӮӢгҒҹгӮҒгҖҒгҒ“гҒЁгҒӘгӮӢгғүгӮӯгғҘгғЎгғігғҲгӮ’еүҠйҷӨгҒ—гҖҒзҹӣзӣҫгҒ—гҒҹгғ¬гғ—гғӘгӮ«пјҲгғҮгғјгӮҝгҒ®з ҙжҗҚпјүгӮ’еј•гҒҚиө·гҒ“гҒ—гҒҫгҒ—гҒҹгҖӮ
  • гӮўгғғгғ—гӮ°гғ¬гғјгғүгҒҢдёҚе®үе®ҡгҒ«гҒӘгӮҠгҒҫгҒҷгҖӮгҒ“гӮҢгҒҜгҖҒDelete by QueryгғӘгӮҜгӮЁгӮ№гғҲгҒҢгғҲгғ©гғігӮ¶гӮҜгӮ·гғ§гғігғӯгӮ°гҒ®дёӯгҒ«гӮҜгӮЁгғӘгҒЁгҒ—гҒҰж®ӢгӮӢгҒҹгӮҒгҒ§гҒҷгҖӮгҒқгҒ®гҒҹгӮҒгҖҒгӮўгғғгғ—гӮ°гғ¬гғјгғүгҒ®гҒӮгҒЁгҒ«жӯЈзўәгҒ«гғ‘гғјгӮ№гҒ•гӮҢгҒӘгҒӢгҒЈгҒҹгӮҠжӯЈзўәгҒ«е®ҹиЎҢгҒ•гӮҢгҒӘгҒ„гҒӢгӮӮгҒ—гӮҢгҒҫгҒӣгӮ“гҖӮдҫӢгҒҲгҒ°гҖҒгӮӨгғігғҮгғғгӮҜгӮ№гӮЁгӮӨгғӘгӮўгӮ№гҒ«еҜҫгҒҷгӮӢгғӘгӮҜгӮЁгӮ№гғҲгҒ§гҖҒгҒқгӮҢгҒҢеүҠйҷӨгҒ•гӮҢгҒҹеҫҢгҒ®е ҙеҗҲгҒ«гҒ“гҒ®гӮҲгҒҶгҒӘгғҗгӮ°гҒҢзҷәз”ҹгҒ—гҒҫгҒҷгҖӮ

еҜҫз…§зҡ„гҒ«гҖҒж–°гҒ—гҒ„гғ—гғ©гӮ°гӮӨгғігҒҜгҖҒе®үе…ЁгҒӘе®ҹиЈ…гҒ§гҒҷгҖӮ scanгҒЁscrollгғӘгӮҜгӮЁгӮ№гғҲгҒ§гӮҜгӮЁгғӘгҒ«гғһгғғгғҒгҒ—гҒҹIDгӮ’иҰӢгҒӨгҒ‘гҖҒгҒқгҒ®IDгӮ’дҪҝгҒЈгҒҰгҖҒbulk indexing APIгҒ§еүҠйҷӨгҒ—гҒҫгҒҷгҖӮ

гҒ“гҒ®е®ҹиЈ…гҒҜгҖҒйҒ…гҒ„еҝ…иҰҒгҒҢгҒӮгӮҠгҒҫгҒҷгҖӮзү№гҒ«гҖҒгӮҜгӮЁгғӘгҒҢеӨҡгҒҸгҒ®гғүгӮӯгғҘгғЎгғігғҲгӮ’еүҠйҷӨгҒҷгӮӢе ҙеҗҲгҒ§гҒҷгҖӮ гӮӮгҒ—гҖҒеӨҡгҒҸгҒ®гғүгӮӯгғҘгғЎгғігғҲгӮ’гҒ“гҒ®APIгӮ’еҲ©з”ЁгҒ—гҒҰеүҠйҷӨгҒҷгӮӢе ҙеҗҲгҖҒгӮўгғ—гғӘгӮұгғјгӮ·гғ§гғігӮ’гғҶгӮ№гғҲгҒ—гҒҰгҒҸгҒ гҒ•гҒ„гҖӮ гҒқгҒ—гҒҰгҖҒд»ЈгӮҸгӮҠгҒ«гӮӨгғігғҮгғғгӮҜгӮ№е…ЁдҪ“гӮ’ж¶ҲгҒҷгӮҲгҒҶгҒӘгӮўгғ—гғӯгғјгғҒгҒ«еҲҮгӮҠжӣҝгҒҲгӮӢгҒ“гҒЁгҒҢгҒ§гҒҚгҒӘгҒ„гҒӢжӨңиЁҺгҒ—гҒҰгҒҸгҒ гҒ•гҒ„гҖӮ

Delete by Query pluginгҒ®гғүгӮӯгғҘгғЎгғігғҲгҒ«гҖҒж–°гҒ—гҒ„е®ҹиЈ…гҒ«гҒӨгҒ„гҒҰгҒ®йҒ•гҒ„гҒӘгҒ©гҒ®гӮҲгӮҠи©ігҒ—гҒ„иӘ¬жҳҺгҒҢгҒӮгӮҠгҒҫгҒҷгҖӮ

Elasticsearch coreгӮ’жңҖе°ҸйҷҗгҒ«

гғ—гғ©гӮ°гӮӨгғігҒ«еҲҮгӮҠжӣҝгҒҲгӮӢгҒ“гҒЁгҒҜгҖҒз°ЎеҚҳгҒӘжұәж–ӯгҒ§гҒҜгҒӮгӮҠгҒҫгҒӣгӮ“гҒ§гҒ—гҒҹгҖӮ еӨҡгҒҸгҒ®гғҰгғјгӮ¶гҒҜе•ҸйЎҢгҒӘгҒҸгҖҒDelete by QueryгӮ’еҲ©з”ЁгҒ—гҒҰгҒ„гҒҫгҒ—гҒҹгҖӮ гҒ—гҒӢгҒ—гҖҒеҚұйҷәгҒҢеёёгҒ«гҒқгҒ“гҒ«гҒӮгӮҠгҖҒдәӣзҙ°гҒЁгҒҜиЁҖгҒ„еҲҮгӮҢгҒӘгҒ„ж•°гҒ®гғҰгғјгӮ¶гҒҢдёҠиЁҳгҒ®гӮҲгҒҶгҒӘж·ұеҲ»гҒӘе•ҸйЎҢгҒ«йҒӯйҒҮгҒ—гҒҰгҒ„гҒҫгҒ—гҒҹгҖӮ

гҒ•гӮүгҒ«гҖҒElsticsearchгҒ®гӮігӮўгҒҜдҝЎй јгҒ§гҒҚгӮӢгӮӮгҒ®гҒ§гҒӘгҒ‘гӮҢгҒ°гҒӘгӮҠгҒҫгҒӣгӮ“гҖӮ д»–гҒ®гӮігӮўAPIгӮ’еҲ©з”ЁгҒ—гҒҰе®ҹиЈ…гҒ§гҒҚгӮӢж©ҹиғҪгҒҜгҖҒгӮігӮўгҒ«еҗ«гҒҝгҒҫгҒӣгӮ“гҖӮзү№гҒ«гҖҒгҒқгӮҢгҒҢгғҗгӮ°гӮ’еҗ«гӮ“гҒ§гҒ„гӮӢе ҙеҗҲгҖӮ гӮігӮўгҒ®гҒҷгҒ№гҒҰгҒ®ж©ҹиғҪгҒҜеј·еӣәгҒ§гҒӮгӮӢгҒ№гҒҚгҒ§гҖҒDelete by QueryгҒҜдәәж°—гҒҢгҒӮгӮҠгҖҒй«ҳжҖ§иғҪгҒ§гҒҷгҒҢгҖҒгҒқгҒҶгҒ§гҒҜгҒӮгӮҠгҒҫгҒӣгӮ“гҒ§гҒ—гҒҹгҖӮ

еҝ…иҰҒгҒ«еҝңгҒҳгҒҰгҖҒгҒ“гҒ®гӮҲгҒҶгҒӘйӣЈгҒ—гҒ„гғҲгғ¬гғјгғүгӮӘгғ•гҒ®жң«гҖҒдҝЎй јжҖ§гҒЁе“ҒиіӘгӮ’йҒёгҒігҒҫгҒҷгҖӮ

гғһгғғгғ”гғігӮ°гҒ®еүҠйҷӨгҒ®е»ғжӯў

гӮҝгӮӨгғ—гҒ®гғһгғғгғ”гғігӮ°гӮ’еүҠйҷӨгҒҷгӮӢж©ҹиғҪгӮӮ2.0гҒ§е»ғжӯўгҒ•гӮҢгҒҫгҒҷгҖӮ гҒ“гӮҢгҒҜгҖҒеҗҢгҒҳгғ•гӮЈгғјгғ«гғүеҗҚгӮ’гҖҒз•°гҒӘгӮӢгғ•гӮЈгғјгғ«гғүгҒ®гӮҝгӮӨгғ—гҒ§еҶҚеҲ©з”ЁгҒ—гҒҹе ҙеҗҲгҒ«гҖҒгӮӨгғігғҮгғғгӮҜгӮ№гҒ®з ҙжҗҚгӮ’еј•гҒҚиө·гҒ“гҒҷеҸҜиғҪжҖ§гҒҢгҒӮгӮӢгҒҹгӮҒгҒ§гҒҷгҖӮ

гҒ—гҒӢгҒ—гҖҒMatch All QueryгҒ§гҖҒDelete by Queryгғ—гғ©гӮ°гӮӨгғігҒ«еҜҫгҒ—гҒҰгӮҝгӮӨгғ—гӮ’жҢҮе®ҡгҒҷгӮӢгҒ“гҒЁгҒ§гҖҒгӮҝгӮӨгғ—гҒ®гҒҷгҒ№гҒҰгҒ®гғүгӮӯгғҘгғЎгғігғҲгӮ’еүҠйҷӨгҒҷгӮӢгҒ“гҒЁгҒҜгҒ§гҒҚгҒҫгҒҷгҖӮ гҒҫгҒҹгҒҜгҖҒ1гҒӨгҒ®гӮӨгғігғҮгғғгӮҜгӮ№гҒ«з•°гҒӘгӮӢгӮҝгӮӨгғ—гӮ’иӨҮж•°еҗ«гӮҒгӮӢд»ЈгӮҸгӮҠгҒ«гҖҒеҖӢеҲҘгҒ®гӮӨгғігғҮгғғгӮҜгӮ№гҒ«еҲҶеүІгҒҷгӮӢгӮҲгҒҶгҒӘгӮўгғ—гғӯгғјгғҒгҒ«еӨүжӣҙгҒҷгӮӢгҒ“гҒЁгӮ’жӨңиЁҺгҒ—гҒҰгҒҸгҒ гҒ•гҒ„гҖӮ
